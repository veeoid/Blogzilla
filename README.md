# Blogzilla

Blogzilla is a simple, lightweight blogging platform designed for ease of use and customization. It allows users to create, manage, and publish blog posts with minimal effort.

## Features
- User-friendly interface for creating and editing blog posts
- Support for Markdown formatting
- Customizable themes and layouts
- SEO-friendly URLs and metadata
- Built-in commenting system


## Installation
1. Clone the repository:
   ```
   git clone https://github.com/yourusername/blogzilla.git
    ```

2. Navigate to the project directory:
    ```
    cd blogzilla
    ```

3. Install the required dependencies:
    ```
    npm install
    ```

4. Start the development server:
    ```
    npm start
    ``` 

5. Setup a port in .env file:
    ```
    PORT=3000
    ```

6. Open your browser and go to `http://localhost:3000` to access the application.


## Usage
- To create a new blog post, click on the "New Post" button and fill in the title and content fields.
- Use Markdown syntax to format your posts.
- Customize the appearance of your blog by selecting a theme from the settings menu.
- Manage comments and moderate user interactions through the admin panel.
